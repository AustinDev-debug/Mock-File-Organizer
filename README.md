# File Organizer (Learning Project)

## Overview

File Organizer is a C# console application created as part of my programming journey. It allows users to create a simple catalog of files, organize them by category, search for saved entries, and store the information using text files.

This project was my **first experience working with the `System.IO` namespace**, including reading from and writing to files.

## Features

* Add file entries
* Organize files by category
* Search for files by name
* Save data to a text file
* Load and display saved data
* Select between the Desktop or Documents directory for storage

## Technologies Used

* C#
* .NET
* System.IO
* Lists
* Classes & Constructors
* Exception Handling
* File and Directory Management

## Learning Notes

This project is not intended to be a production-ready file organizer or password manager. It was built to help me learn how file handling works in C#.

I know there are areas that could be improved, such as file formatting, searching, and loading data more efficiently. Rather than trying to make everything perfect on my first attempt, I focused on understanding the core concepts behind `System.IO`.

As I continue learning C#, I plan to revisit this project and improve it using better practices and more advanced techniques.

## Purpose

The goal of this project was to gain hands-on experience with:

* Reading and writing text files
* Creating files if they don't exist
* Working with directories
* Using `StreamReader` and `StreamWriter`
* Handling file-related exceptions
* Building applications that can save data between program sessions
